# QQ 邮箱授权码获取完整教程

## 步骤 1：登录 QQ 邮箱

1. 打开浏览器访问 https://mail.qq.com
2. 使用 QQ 账号和密码登录

## 步骤 2：进入设置

1. 点击顶部 **设置** → **账户**

## 步骤 3：开启 SMTP/IMAP 服务

找到 **"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"** 部分

开启以下选项：
- ✅ **IMAP/SMTP服务**
  - 勾选 "IMAP/SMTP服务"
  - 点击 "生成授权码" 按钮
  - 按提示发送短信到指定号码
  - 获得 **16位授权码**（格式类似：`abcdxyz123456789`）

- ✅ **POP3/SMTP服务**（可选，但建议开启）
  - 同样需要生成授权码

## 步骤 4：复制授权码

- 授权码是 **16位字母数字组合**
- ⚠️ **不是 QQ 密码！**
- ⚠️ **不是短信验证码！**
- ✅ 是类似：`XXXXXXXXXXXXXX` 的字符串

## 步骤 5：更新 .env 文件

```bash
# 打开 .env 文件
# 将 EMAIL_PASSWORD 的值替换为你的16位授权码

EMAIL_PASSWORD=你的16位授权码
```

## 步骤 6：验证

```bash
cd D:\code\Horizon
uv run python test_smtp.py
```

如果看到 ✅ 登录成功，说明配置正确！

## 常见问题

### Q1: 找不到 "生成授权码" 按钮？
- 确保已开启 IMAP/SMTP 服务
- 某些账号可能需要先在 "安全设置" 中验证

### Q2: 发送短信后没收到授权码？
- 检查是否发送到正确的号码
- 授权码通常在网页上直接显示，不会发送到手机

### Q3: 授权码一直登录失败？
- 重新生成一个新的授权码
- 确保复制时没有多余的空格
- 尝试重启 Horizon 服务

### Q4: 提示"请先开启IMAP服务"？
- 在邮箱设置中确认 IMAP/SMTP 服务已勾选
- 点击 "保存更改"

## 安全建议

- ✅ 授权码妥善保管，不要泄露
- ✅ 不要将授权码提交到 Git 仓库
- ✅ 可以随时在邮箱设置中删除旧授权码
- ✅ 建议定期更换授权码
